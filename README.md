> The Simple and Free MacOS Color Picker App

__Requirements__: MacOS `12.0` (Monterey) Or Above 

<img src="https://raw.githubusercontent.com/problaze20/Hexor-Utility/refs/heads/main/Images/Image-v1.0.7.png" alt="Screenshot" width="600" />

## Features
- <kbd>Cmd + P</kbd> to __pick a color from your screen__.
- Use <kbd>Cmd + Opt + R</kbd> to __generate a random color__.  
- Compatible with both __Intel__ and __Apple Silicon__ Macs.  
- Copy colors in multiple formats:
  - **App frameworks:** SwiftUI RGB, Flutter HEX / RGB  
  - **Web:** CSS RGB/HSL, HEX, HSL  
  - **Advanced:** HSV, LCH, LAB, CMYK, XYZ
